
A chatbot interface built as a practice project from the SuperSimpleDev React course.

## About
A simple chatbot that can answer basic questions about the date, time, and weather.
Responses are powered by SuperSimpleDev's chatbot library.

## What I practiced
- useState for managing chat messages
- Controlled inputs with onChange
- Passing state between components via props
- useRef + useEffect for auto-scroll behavior

## Tech
HTML, CSS, React (CDN), SuperSimpleDev Chatbot Library
